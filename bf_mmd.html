HOW TO DOWNLOAD MODELS?

When you open the link you'll probably get to a Nico video, Nico image, Blomaga Article, bilibili video or a Tweet.

Finding the DL link and the password is quite simple:

  • The DL link is, well, you know, a link (like https://bowlroll.net/file/randomnumbers).
  • The password is a word, code or sentence, usually next to the link.
  • The password hint is usually in Japanese, if you want to download a Japanese person's model. Sometimes the password hint is connected to the video, so you need to be logged in to watch it.
  • Distribution in community:
    The link for the community is also on the list. After joining the community, you can find the DL link and password on the Community Board. If the community is marked with (Japanese), then you have to write a message to the modeler in Japanese (No Google Translate!!) to get in the community.
  • Distribution on Twitter list:
    You need to message the modeler on Twitter and ask them to add you to their Twitter list.
    After being added, you will be able to view a Privatter post that only list members can and you will find the DL link/password there.
  • Distribution in Twitter DM:
    You need to message the modeler on Twitter and ask them to give you the DL link/password.

When you open the Bowlroll (or other) link, you have to put the password there (if needed) and click Download. It's so simple!

The distribution was DL-number limited and the file isn't available anymore.

This can mean that it's a private distribution or the distribution stopped.

WHAT TO DO WITH THE DOWNLOADED FILE?

Right-click on it and select "Properties" option in the list. Click on "Unblock", then confirm by "Apply", then "OK".

There are 2 ways of opening the zip/rar file:

  1. Set your computer's locale to Japanese before you extract the file (or Chinese if the modeler is Chinese!), so your comp won't mess up the texture names and the model won't look weird (white).
    Then use any file extracting software of your choice (Windows File Explorer, Total Commander, etc.)
     
  2. Download Bandizip, install it and configure it before using.
    Open the zip/rar file in it, change the codec to Japanese (or Chinese if the model is by a Chinese person, etc.) and unzip the file now.

    Note: While file names will be OK after this, Japanese or Chinese language ReadMe's will still look weird.
    In order to be able to read it, open them in Notepad++, click Encoding, then Character sets, then select the Japanese or Chinese one.

If there's another zipped folder in the original one, you probably need another password for it. The pass or its hint is usually in a ReadMe file.

Read the ReadMe to know what you're allowed to do with the models!
Please don't anger the modelers by doing things with their models that aren't allowed.
